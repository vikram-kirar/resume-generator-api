# Privacy Policy

This GPT uses an external API to generate resume PDFs.

## Data Collected

The following information may be sent to the API when generating a resume:

* Name
* Email
* Phone
* LinkedIn
* Work experience
* Education
* Skills

## How Data Is Used

The data is used only to generate a resume PDF and return a download link.

## Data Storage

The generated resume file may be temporarily stored on the server to allow downloading.

## Data Sharing

No personal data is sold, shared, or used for marketing.

## Contact

For questions about this privacy policy, contact the developer.
